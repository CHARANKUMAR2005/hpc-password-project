What is this project?

This project demonstrates Parallel Password Cracking using the concept of High Performance Computing (HPC). It is only a learning simulation, not real hacking.

How it works (Simple English)

  • A sample password is already fixed inside the program.
  • The system tries many password combinations.
  • Multiple CPU cores work at the same time (parallel processing).
  • The correct password is found faster.

Why HPC is used?

HPC helps reduce time by dividing work among multiple CPU cores. This improves speed and efficiency compared to single-core execution.

Important Note

This project is strictly for educational purposes. It does not crack real passwords and does not access any real accounts.
